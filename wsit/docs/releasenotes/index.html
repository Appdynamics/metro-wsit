Metro ${IMPL_VERSION} ${IMPL_VERSION_SUFFIX} is a Web Services framework that provides tools and infrastructure to develop Web Services solutions for the end users and middleware developers. With Metro, clients and web services have a big advantage: the platform independence of the Java programming language.
